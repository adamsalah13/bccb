# API Application

## Backend for BCCB Micro-Credentials Platform

Node.js + TypeScript REST API with AI integration.

## Features

- RESTful API endpoints
- PostgreSQL + Prisma ORM
- Redis caching
- Bull queue for async jobs
- AI/ML service integration
- JWT authentication

## Setup

```bash
npm install
npm run dev
```

## Tech Stack

- Node.js 20+
- Express/Fastify
- TypeScript
- Prisma ORM
- PostgreSQL
- Redis
- Bull Queue

# Web Application

## Frontend for BCCB Micro-Credentials Platform

React 18 + TypeScript + Vite application with AI-native interfaces.

## Features

- Conversational form interfaces
- Mermaid diagram visualization
- Real-time AI recommendations
- Responsive design with TailwindCSS

## Setup

```bash
npm install
npm run dev
```

## Tech Stack

- React 18
- TypeScript
- Vite
- TailwindCSS
- Redux Toolkit
- React Query
- Mermaid.js
